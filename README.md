# Rivot Website

A full-stack website project containing frontend pages, admin panel, and payment processing modules.

## 🚀 Features

* Dynamic website pages
* Admin dashboard for managing items
* Payment integration
* Secure authentication system
* Responsive UI design

## 🛠️ Technologies Used

* PHP
* HTML5
* CSS3
* JavaScript
* MySQL
* Bootstrap

## 📁 Project Structure

```
rivot/
│
├── site_admin/        # Admin panel
├── process_payment/   # Payment processing module
├── zaak/              # Payment API integration
├── phpMyAdmin/        # Database management files
├── assets/            # CSS, JS, images
└── index.php          # Main website entry
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/sahabiswa96/rivot-website.git 
```

2. Move the project to your server directory (XAMPP / WAMP / LAMP)

3. Import the database in **phpMyAdmin**

4. Run the project in browser
